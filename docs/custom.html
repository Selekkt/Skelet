CSS vars

To simplify things a lot and to keep consistency throughout Skelet.css, we use CSS var().

Skelet. var()

On the left is a list of vars thay you can copy and paste into your CSS file to change the colors font sizes, paddings, margins throughout Skelet.

Inline var()

You can also use these vars inline directly on your element so you can define styles per element.

Animations are a good example. Skelet previously used classes to define animation duration and delay; however, you were limited to the defined classes.

You can now set your own animation duration and delay by doing the following:

<h1 start animation=fadein ease=easeInOutQuad style="--duration: 100ms">
<h1 start animation=fadein ease=easeInOutQuad style="--duration: 100ms">
