Accessibility

Accessibility (element outline) can be activated on all elements supporting it by adding class="a11y" to your <body class="a11y"> or just to a specific element: <footer class="a11y">.

.a11y Default one using browsers defaults. .a11y--custom Customizable one using box-shadow 
:root {
    --a11yStyle: 0 0 0 0.2rem;
    --a11yColor: rgba(0, 126, 252, 0.6);
    /* ^ this will convert in: box-shadow: 0 0 0 0.2rem rgba(0, 126, 252, 0.6); */
}

Since we're using "box-shadow" for .a11y--custom you can easily personalize the look of the outline.

 Dark mode

In case you're building a Dark website, you can set Skelet. automatically in dark mode, doing so you'll have everything set in darker shades instead of light.

To activate "Dark mode" you need to add one of the options below to <body> tag

 dark-mode

This is way two of activating Dark mode

 data-mode="dark"

This is one way of activating Dark mode

 .dark-mode

This is way three of activating Dark mode

Scroll snap .scroll-snap This feature allows you to create well-controlled scroll experiences by declaring scroll snapping positions.   .y The scroll container snaps in its vertical axis only. .mandatory This property enables specific strict value that go to the specific scroll position when there is no scrolling. .proximity This property enables specific strict value that go to the specific scroll position. If scrolling action gets pretty close to a snap position if not close then it will act normally and should not snap at all. .align-start The start and end alignment of this box's scroll snap area, within the scroll container's snapport is a snap position in this axis. .align-center The center alignment of this box's scroll snap area, within the scroll container's snapport is a snap position in this axis. .align-end The end alignment of this box's scroll snap area, within the scroll container's snapport is a snap position in this axis. .stop-always The scroll container must not "pass over" a possible snap position; and must snap to the first of this elements' snap positions.

Tooltips

To use Tooltips simply include the https://cdn.jsdelivr.net/gh/Selekkt/skelet@next/css/skelet-tooltips.css in your file.

Then add your text inside data-tooltip and use one of the tooltip-pos from the list below to position your tooltip. You can also control its dimensions.
