Flex direction

You can decide the direction of your flexbox, if it's going to be a column or a row <x-flex direction=*>

 Available options
  • direction=column
  • direction=column-reverse
  • direction=row
  • direction=row-reverse

Wrap

Specifies whether flex items are forced into a single line or can be wrapped onto multiple lines. If wrapping is allowed, this property also enables you to control the direction in which lines are stacked. wrap=*

 Available options
  • wrap
  • wrap=nowrap
  • wrap=reverse

Visual order

Specifies the order in which flex items are laid out in their flex container. Elements are arranged in ascending order of order value. Elements with the same order value are displayed in the same order as they appear in the source code. To make this work, all elements must have order=5 set to them. Aside from order=first and order=last, numbers from 1 to 16 can be used.

 Available options
  • order=first
  • order=last
  •  
  • order-s=first
  • order-s=last
  •  
  • order-l=first
  • order-l=last

Justification

Defines how the browser distributes space between and around flex items along the main-axis of their container. jc=*

 Available options
  • jc=start
  • jc=center
  • jc=end
  • jc=between
  • jc=around

Align content

Aligns a flex container's lines within the flex container when there is extra space on the cross-axis. ac=*

 Available options
  • ac=start
  • ac=center
  • ac=end
  • ac=between
  • ac=around
  • ac=stretch

Align items

Aligns flex items of the current flex line the same way as justify-content but in the perpendicular direction. ai=*

 Available options
  • ai=start
  • ai=center
  • ai=end
  • ai=baseline
  • ai=stretch

Align self

To be used on <x-cell> Aligns flex items of the current flex line overriding the align-items value. as=*

 Available options
  • as=auto
  • as=start
  • as=center
  • as=end
  • as=baseline
  • as=stretch
